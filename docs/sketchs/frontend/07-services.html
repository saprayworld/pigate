🧠 Backend Integration Concept สำหรับ nftables

แนวคิดเดียวกับหน้า Addresses ครับ แนะนำให้ใช้คุณสมบัติ Named Sets ของ nftables ในการจัดกลุ่มพอร์ต โดยหลังบ้าน (Python) จะสร้าง Set สำหรับพอร์ตขึ้นมาแยกตามโปรโตคอล ตัวอย่างโครงสร้างคำสั่งที่จะใช้ส่งไปคุมระบบบน Raspi 5 มีลักษณะดังนี้:

# 1. สั่งสร้างกลุ่มตัวแปร (Set) สำหรับพอร์ต TCP ไว้รอในตาราง nft add set ip filter Web_Testing_Pool { type inet_service; } # 2. ทำการเพิ่มช่วงพอร์ตเข้าไปในตัวแปรนั้น nft add element ip filter Web_Testing_Pool { 8080-8085 } # 3. เวลาเขียนกฎในหน้า Firewall Policy ก็อ้างอิงชื่อตัวแปรผ่านเครื่องหมาย @ ได้เลย nft add rule ip filter FORWARD tcp dport @Web_Testing_Pool accept
