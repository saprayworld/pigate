🧠 Backend Integration สําหรับ หน้า Dashboard (กลไกการดึงข้อมูล Real-time)

เพื่อให้หน้า Dashboard แสดงผลแบบขยับได้สดๆ บนหน้าเว็บ แนะนำให้เขียนฝั่ง Frontend ยิงคำสั่งดึงข้อมูล (Polling API) มาอัปเดตทุกๆ 3-5 วินาที โดย Backend (Python) จะใช้ช่องทางของคอร์ระบบในการไปแกะข้อมูลมาจัดฟอร์แมต JSON:

# 1. การดึงค่า CPU / RAM / Temp (ใช้ Library 'psutil' ใน Python จะง่ายที่สุด) import psutil cpu = psutil.cpu_percent() ram = psutil.virtual_memory().percent # ดึงความร้อนบอร์ด RPI5 with open("/sys/class/thermal/thermal_zone0/temp", "r") as f: temp = int(f.read()) / 1000 # 2. การดึงประวัติการบล็อกคัดกรองแพ็กเก็ตจาก nftables มาแสดงในตาราง Log # ต้องเปิดการทำ log ในกฎ nftables ก่อน (เช่น nft add rule ip filter FORWARD log prefix "PiGate_Drop: " drop) # จากนั้นดึงข้อความล่าสุุดจากระบบ dmesg หรือ syslog ออกมาวิเคราะห์ sudo dmesg | grep "PiGate_Drop" | tail -n 5
